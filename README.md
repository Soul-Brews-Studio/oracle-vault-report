# Oracle Vault Report

OLED dark-mode dashboard for the Oracle ψ/ memory structure.

**[Live Demo](https://soul-brews-studio.github.io/oracle-vault-report/)**

## What it shows

- `.md` file counts across all ψ/ repos
- Vault project distribution by org
- Symlinked vs real ψ/ breakdown
- Rsync-eligible repos
- Oracle skills inventory

## Usage

```bash
# Generate report from your machine
node generate.mjs

# Generate + commit + push (updates GitHub Pages)
node generate.mjs --push
```

### Requirements

- Node.js 18+
- [ghq](https://github.com/x-motemen/ghq) for repo management
- Repos with `ψ/` directories (Oracle memory structure)

### Vault detection

1. `$ORACLE_VAULT_PATH` env var
2. ghq repo matching `*oracle-vault`
3. No vault = shows ψ/ scan only

## No secrets

Only repo names, file counts, sizes, and skill names are included. No file contents, paths, or credentials.
